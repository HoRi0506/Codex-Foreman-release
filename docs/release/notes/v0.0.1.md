# v0.0.1

- resets the public release history to a fresh Rust-only baseline
- ships the Foreman runtime as a Rust binary bundle instead of an npm install surface
- installs the public `$cap` skill from the packaged release bundle
- keeps the captain-first checklist model as the public interaction contract
- fixes delegated worker launch so child `codex exec` does not inherit stdin and stall
- moves the shared Foreman config surface to `foreman-config.toml` and keeps legacy JSON as a migration fallback
- applies shared role model/profile/reasoning settings to delegated `codex exec` launches instead of ignoring them at runtime
- removes the ordinary manual `codex_bin` requirement by resolving the effective Codex binary from the run or local environment
- restores richer Rust status/activity visibility for fan-in readiness, active delegations, and best-effort token usage from worker raw events
