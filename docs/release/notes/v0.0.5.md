# v0.0.5-pre

> Pre-release notice: `v0.0.5-pre` is still a beta/pre-release train. Install or update, fully restart Codex CLI, then run `ccc check-install`.

## Theme

`v0.0.5-pre` focuses on captain-constrained harness behavior.

## Exposure Policy

- The public release repo keeps `share/skills/cap/SKILL.md` as a thin placeholder.
- `scripts/build-release-asset.sh` copies the authoritative `$cap` skill from the source repo into the packaged release asset.
- Public browsing of the release tree should not expose the full installable skill text.
- History rewrite remains deferred; the current release-repo history stays intact until an explicit release decision changes it.

## Release Scope

- no-mutation preflight before default run creation
- MCP as a supported control-plane surface
- compact status as the allowed-action contract
- captain action guard for direct mutation, finish, review skip, and fallback
- mandatory fan-in before finish or follow-up
- visible direct-fallback and degraded-path history
- reduced `$cap` prompt surface with runtime/MCP-owned durable policy

## Asset Scope

- Published asset: `ccc-0.0.5-pre-darwin-arm64.tar.gz`
- Linux, Windows, and other platform assets are not published for this pre-release yet.

## Validation

- `cargo test -p ccc --offline`: passed
- `./scripts/check-release-sensitive-strings.sh`: passed
- `./scripts/verify-release-asset-matrix.sh`: passed
- `./scripts/verify-install-pruning.sh`: passed
- Real asset inspection confirmed the packaged `share/skills/cap/SKILL.md` matches the source repo skill and differs from the public placeholder.

## Upgrade Notes

- Fully restart Codex CLI after install/update.
- Run `ccc check-install` in the restarted session.
- History rewrite remains deferred; the public release tree now carries a placeholder skill while the installable asset carries the authoritative source skill.
